# Configuration Files

This directory contains YAML configuration files for the orchestrator.

## Files

- `orchestrator-config.yaml` - Main configuration file (will be created in Phase 4)
- `orchestrator-config.local.yaml` - User-specific overrides (gitignored)

## Configuration Options

The configuration system will support:
- Model selection and parameters
- Retry limits and verification thresholds
- Branch management preferences
- Artifact storage locations
- Prompt templates and customization
- RAG system settings

See Phase 4 documentation for detailed configuration options.
