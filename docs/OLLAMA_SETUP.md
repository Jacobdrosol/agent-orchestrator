## Quick Installation Steps (Preliminary)

1. Download Ollama for Windows from the official website
2. Install Ollama following the installer prompts
3. Open PowerShell and run: `ollama pull qwen2.5-coder:14b-instruct-q4_K_M`
4. Verify installation: `ollama list`
