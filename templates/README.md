# Jinja2 Templates

This directory contains Jinja2 templates for generating specs, findings reports, and prompts.

## Template Categories

### Spec Templates
- Phase spec generation
- Implementation details
- Task decomposition
- Verification criteria

### Findings Templates
- Test results reporting
- Lint and security scan summaries
- Build status reports
- Comprehensive findings aggregation

### Prompt Templates
- LLM system prompts
- Context injection templates
- RAG retrieval formatting
- Conversation management

## Usage

Templates will be rendered using Jinja2 with context data from:
- Repository analysis (RAG system)
- Phase planning state
- Verification results
- Configuration settings

Templates will be created in Phase 8.
